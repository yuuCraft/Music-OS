# Music OS　V-1.00.0
## Webであなたの音楽を保存。SONY・Xアプリの後継を目指すソフトです。
<img src="https://copilot.microsoft.com/th/id/BCO.84de6e7c-a21d-44aa-b8ca-d97e39532d56.png" alt="A sleek, modern logo for &#39;Music OS&#39; featuring a stylized musical note integrated with a circular wave design, using blue gradient tones and minimalist Japanese-inspired typography. Make the overall logo smaller and more compact."/>

**《主な機能》**
- 洗練されたGUI
- アーティスト・プレイリストをID3 Tagで分類
- 背景・高音低音調整など、カスタマイズ性抜群
- バックアップ機能で、WALKMAN等ミニ音楽プレイヤーの音楽導入をスムーズに。

**《内部構成》**
- HTML・CSS・Javascriptで構成。
- HTMLファイルだけをダウンロードすると、オフラインでも使用できます。PC推奨です。
- 機能を正常にご利用いただくためには、Chromiumベースブラウザ（Chrome、Edge等）使用を推奨します。

**《ライセンス》**
- HTMLによるオフライン使用時のみ、ファイルダウンロードを許可します。
